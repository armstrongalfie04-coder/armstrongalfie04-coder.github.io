Data Model

From raw files to clean reporting tables

The prototype separates imported CSV files from validated final tables. That keeps mistakes traceable while still giving the appeal team clean data for reporting.

  • Staging tables preserve the original source rows.
  • Validation errors record rejected values and field-level messages.
  • Lookup tables keep roles, statuses, payment methods, and expense types consistent.
  • Final tables support volunteers, events, assignments, donations, and expenses.
